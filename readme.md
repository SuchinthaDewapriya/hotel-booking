## 🇺🇸 English

### 🚀 Overview
A professional web application for hotel room reservations built with **Laravel**. This project demonstrates full-stack development capabilities, including database management and administrative controls.

### ✨ Key Features
* **User Authentication:** Secure login and registration for customers.
* **Room Management:** Real-time availability check with pricing details.
* **Booking System:** Seamless reservation process with date selection.
* **Admin Dashboard:** Comprehensive tools to manage bookings, rooms, and users.
* **Search & Filters:** Advanced search for rooms based on user requirements.

### 🛠 Tech Stack
* **Backend:** Laravel 10/11 (PHP)
* **Database:** MySQL
* **Frontend:** Blade Templates, Bootstrap, JavaScript
* **Tools:** Composer, Git/GitHub

## 🇯🇵 日本語 (Japanese)

### 🚀 概要
**Laravel** で構築された、プロフェッショナルなホテル客室予約システムです。このプロジェクトは、データベース管理や管理者用ダッシュボードを含む、フルスタックの開発能力を証明するものです。

### ✨ 主な機能
* **ユーザー認証:** 顧客向けの安全なログインおよび新規登録機能。
* **客室管理:** 価格や詳細情報を含む、リアルタイムの空室確認。
* **予約システム:** 日付選択から完了までスムーズな予約フロー。
* **管理者ダッシュボード:** 予約、客室、ユーザー情報を一括管理。
* **検索・フィルタリング:** 条件に合わせた客室の高度な検索機能。

### 🛠 使用技術
* **バックエンド:** Laravel 10/11 (PHP)
* **データベース:** MySQL
* **フロントエンド:** Blade テンプレート, Bootstrap, JavaScript
* **ツール:** Composer, Git/GitHub

---

### ⚙️ Installation / インストール
1. `git clone https://github.com/SuchinthaDewapriya/hotel-booking.git`
2. `composer install`
3. Copy `.env.example` to `.env` and configure your database.
4. `php artisan migrate`
5. `php artisan serve`
